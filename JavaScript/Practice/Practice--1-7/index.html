Practice Questions

  1. Q1. Write a function that evaluates any math expression string manually without using eval().
  2. Q2. Create a function deepFreeze(obj) that prevents modifications in nested objects as well.
  3. Q3. Implement a function that returns the most repeated character in a string.
  4. Q4. Write a function that prints numbers from 1 to 200 but skips all numbers that have the digit 7.
  5. Q5. Create a function that takes an object and returns a new object that contains only properties whose values are functions.
  6. Q6. Implement a function that accepts an array of objects and returns the object with maximum key count.
  7. Q7. Write a function that behaves like Array.map() but using only loops.
  8. Q8. Create a function that reverses an array without using reverse(), push(), or pop().
  9. Q9. Implement a function validateUser(userObj) where all properties must be strings and not empty.
  10. Q10. Write a function that accepts a number n and generates the first n prime numbers.
  11. Q11. Create a loop that prints all numbers between 1 and 1000 whose sum of digits is even.
  12. Q12. Write a function to flatten a nested array up to any depth without using flat().
  13. Q13. Implement a higher-order function operate(a, b, func) that performs custom operations.
  14. Q14. Create a function that compares two objects deeply and returns true only if all nested values match.
  15. Q15. Write a scheduler that logs “tick” every 500ms and “tock” every 1s and stops both after 6 seconds.
